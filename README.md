# Leul Zemede | Personal Portfolio Website

This is my personal portfolio website - an aspiring **Full Stack Developer** and third-year **Information Systems** student at **Addis Ababa University**.  

It showcases my’s **skills, projects, and professional profile** in a modern, responsive design.

---

## 🌟 Features

- **Professional Hero Section**: Introduction with name and role.
- **About Me**: Detailed description of skills, interests, and approach to problem-solving.
- **Skills Section**: Frontend, backend, and other relevant technologies.
- **Projects Section**: Highlights 3 major projects:
  1. **Personal Portfolio Website** – Showcases skills and projects (HTML, CSS)
  2. **Client Lead Management System** – Mini CRM with frontend and backend (HTML, CSS, JS, Node.js, Express, MongoDB)
  3. **Local Business Website** – Responsive site for a local café (HTML, CSS, JS, WordPress)
- **Contact Section**: Email contact link.
- **Responsive Layout**: Optimized for mobile and desktop.
- **Modern, Interactive Design**: Card hover effects, clean typography, cyber-green accent.

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript, React.js (planned)
- **Backend**: Node.js, Express, Python, SQL, MongoDB
- **Version Control**: Git & GitHub
